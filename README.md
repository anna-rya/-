## Описание

GitHub User Finder — это desktop-приложение с графическим интерфейсом для поиска пользователей GitHub, просмотра их профилей и сохранения избранных пользователей в локальное хранилище.

### Основные возможности

- 🔍 Поиск пользователей GitHub по имени/логину
- 📋 Отображение результатов поиска в удобной таблице
- ⭐ Добавление пользователей в избранное
- 💾 Сохранение избранного в JSON файл
- 🌐 Открытие профилей пользователей в браузере
- 📊 Детальная информация о пользователях (ID, URL)

### Требования

- Python 3.7+
- Библиотеки: `requests`, `tkinter` (входит в стандартную поставку Python)
